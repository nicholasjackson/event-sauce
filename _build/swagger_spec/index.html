Event Sauce API

Event Sauce is a simple Event Sourcing system, the API allows the registration of consumers for events and also allows clients to send an event which is then dispatched to the registered consumers.

post /event
Add a event to the queue (eventPost)
The event endpoint allows messages to be added to the event store, when a message is received it will be queued for dispatch to any consumers.

get /health
Health Check (healthGet)
The Health Check endpoint is used to determine the current status for the health of the api.\nThis endpoint will be used by other systems such as Consul and other service discovery systems.

Responses

200
